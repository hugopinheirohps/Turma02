git init (cria um novo repositório local)
git clone (Clonar um repositório exixtente)
git commit -m "" (Ele leva as mudanças de um ambiente local para o repositório no git)
git add . (adiciona arquivos modificados)
git status (Para saber algumas informações sobre a ramificação na qual você está trabalhando agora)
git pull (Esse comando traz para a sua máquina todas as mudanças que foram realizadas na plataforma.)
git config (O config é um comando inicial para vincular o trabalho no repositório com sua conta no github. Assim, é configurado com o nome e com o e-mail. )
git push (O push serve para subir as alterações de uma ramificação para um certo repositório. Ele entrega todos os commits e a mensagem.)
git merge (para fazer uma conjunçao da branch em que está trabalhando com as outras)
git rm (remove arquivos)
