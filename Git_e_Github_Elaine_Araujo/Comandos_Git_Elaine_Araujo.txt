git init ( Criar um repositório)
git clone ( Clonar um repositório)
git status ( conferir o status das alterações)
git add ( Gravar uma alteração)
git commit -m ( Salvar as alterações com comentários)
git branch ( Criar uma Branch do arquivo principal, cloando ele e alterando sem ser a versão final)
git checkout (Mudar para a Branch)
git Push origin ( transfere commits a partir do repositório local para um repositório remoto )
git Pull origin (Incorpora as alterações de um repositório remoto no branch atual.)
git merge ( Após tudo conluido caso deseje juntar o Branch com o arquivo original, este é inreversivel.)
