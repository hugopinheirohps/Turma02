Criar novo repositório
git init

Verificar estado dos arquivos/diretórios
git status

Adicionar arquivo/diretório (staged area). Adicionar um arquivo em específico
git add meu_arquivo.txt

Adicionar um diretório em específico
git add meu_diretorio

Adicionar todos os arquivos/diretórios
git add .

Comitar um arquivo
git commit meu_arquivo.txt

Comitar vários arquivos
git commit meu_arquivo.txt meu_outro_arquivo.txt

Comitar informando mensagem
git commit meuarquivo.txt -m "minha mensagem de commit"

Remover arquivo
git rm meu_arquivo.txt

Remover diretório
git rm -r diretorio

Exibir histórico
git log

Vincular repositório local com um repositório remoto
git remote add origin git@github.com:leocomelli/curso-git.git

Enviar arquivos/diretórios para o repositório remoto. O primeiro push de um repositório deve conter o nome do repositório remoto e o branch.
git push -u origin master

Clonar um repositório remoto já existente
git clone git@github.com:leocomelli/curso-git.git

Atualizar os arquivos no branch atual
git pull
