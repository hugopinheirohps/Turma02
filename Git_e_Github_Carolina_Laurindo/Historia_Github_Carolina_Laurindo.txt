a. Quem inventou o sistema de controle de versão Git e por quê?
	O git foi inventado por Linus Torvalds após Linus ter ficado insatisfeito com uma mudança de licença de utilização do BitKeeper, o sistema de controle de versão que era utilizado por ele e pela equipe que desenvolvia o Linux.
